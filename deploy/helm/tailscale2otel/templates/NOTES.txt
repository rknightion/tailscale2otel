tailscale2otel has been deployed as {{ include "tailscale2otel.fullname" . }}.

It is a singleton poller that ships Tailscale observability data to your OTLP
endpoint ({{ .Values.config.otlp.endpoint }}) every {{ .Values.config.otlp.metric_interval }}.

1. Confirm it started:
   kubectl logs -l app.kubernetes.io/instance={{ .Release.Name }} -f

2. Credentials:
{{- if .Values.existingSecret }}
   Using existing Secret "{{ .Values.existingSecret }}" — ensure it exposes
   TS2OTEL_TAILSCALE__AUTH__OAUTH__CLIENT_ID / TS2OTEL_TAILSCALE__AUTH__OAUTH__CLIENT_SECRET
   (or TS2OTEL_TAILSCALE__AUTH__APIKEY) and TS2OTEL_OTLP__GRAFANA_CLOUD__INSTANCE_ID /
   TS2OTEL_OTLP__GRAFANA_CLOUD__TOKEN.
{{- else }}
   Rendered into Secret "{{ include "tailscale2otel.fullname" . }}". Set values via
   --set secret.TS2OTEL_TAILSCALE__AUTH__OAUTH__CLIENT_ID=... etc., or point
   existingSecret at your own.
{{- end }}

3. In Grafana Cloud Explore, look for metrics like tailscale.device.online,
   tailscale.network.io, and tailscale2otel.up.
